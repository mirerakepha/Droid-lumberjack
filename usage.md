## Get started locally

#### Clone the repository
```git clone https://github.com/mirerakepha/Droid-lumberjack.git```

#### Then

```cargo run```

#### Here below are the two ways you can use to get it up and running when developing

1. ============================

   Run binary directly in Android Studio
   ```./target/debug/lumberjack start```

   ============================



2. ============================
   
   install and add to path
   ```cargo install --path .```
   then every time you save the project use it 
   ```cargo install --path . --force```

   the command to start lumberjack in Android Studio
   ```lumberjack start```
   
   ============================
